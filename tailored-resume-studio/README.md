# Tailored Resume Studio

This folder contains the local app source.

For the full installation and usage guide, see the repository-level README:

```text
../README.md
```

Quick start from this folder:

```powershell
npm install
npm start
```

Then open:

```text
http://localhost:5178
```

Optional AI configuration belongs in a local `.env` file in the repository root. Do not commit `.env`; use `../.env.example` as the safe template.
